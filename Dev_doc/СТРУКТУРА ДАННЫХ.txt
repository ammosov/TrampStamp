СТРУКТУРА ДАННЫХ

Игра использует два файла данных: 
Уровни: rounds.csv
Карты: cards.csv

Конвенции
1. Последовательная нумерация параметров игроков придерживается списка 
'Voters' : 0, 
'Swing Voters' : 1, 
'Partisans' : 2, 
'News' : 3, 
'Hype' : 4, 
'Cash' : 5, 
'Media' : 6, 
'Mojo' : 7, 
'Money' : 8, 
'all resources' : 9, 
'all assets' : 10, 
'Free turn' : 11,

2. Префиксы
t, 0 = Trump
h,1 = Hillary  
3. Понятия
'resources' = news, hype, cash
'assets' = money, media, mojo

Rounds.csv

Назначение: данные об уровнях
Применение: считывается при активации уровня (установка, открытие описания уровня, запуск уровня)
Содержание: условия игры
- Условия победы
- Условия поражения
- Стартовые позиции игроков
- Картинка игрового поля
- Запрещенные для игры карты (не задействовано)
Описание файла:
Id – UID строки; NUM
destr – достигается ли победа уничтожением voters противника / поражение потерей своих voters; 0 – нет, 1 – да; BOOL
res – количество любого из "ресурсов", накопление которого является условием победы: NUM
turn – первый ход принадлежит 0 – Трампу, 1 – Хиллари; BOOL
t1-t8 – начальные значения игрока 0 (Трампа); NUM
h1-h8 – начальные значения игрока 1 (Хиллари); NUM


Cards.csv – текущая версия

Назначение: данные о картах (принадлежность, действия, изображения)
Применение: считывается на старте уровня 
Примечание: сейчас колоды Трампа и Хиллари тождественны по составу и действиям и отличаются лишь по изображениям
Описание файла:
Id – UID строки; NUM
ttitle – название карты для Трампа, STR
htitle - название карты для Хиллари, STR
cost_color – цвет карты (ресурс, который расходуется на игру ею); 0 – карта бесплатная, 1 – news, 2 – hype, 3 – cash; NUM
cost_value – количество ресурса, необходимое для игры картой; NUM
descr – описание карты; STR (не задействовано; * - код переноса строки)
act1-act3 – каждая карта может совершать от 1 до 3 действий одновременно, каждое действие имеет 3 характеристики: "значение", "тип" и "сторона" 
act1_value – "значение", инкремент параметра, который производит карта; положительное значение – карта увеличивает параметр, отрицательное - уменьшает; NUM
act1_type – "тип действия", т.е. игровой параметр, на который действует карта, значения см. список параметров; NUM
act1_side – "сторона", то есть игрок, на параметры которого карта производит действие (target); 0 – you, 1 - opponent, 2 – both; NUM
img_t – наименование файла с изображением карты Трампа
img_h - наименование файла с изображением карты Хиллари
